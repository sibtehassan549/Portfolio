React.js Node.js MySQL Docker JWT PDFKit Leaflet.js

Developed a full-stack e-governance web application to digitize land ownership records and property transfers in Pakistan, replacing manual paper-based processes.

Key Features:
  • Implemented Role-Based Access Control (RBAC) with 6 distinct user roles (Citizens, PLRA Officers, Patwari, Tehsil, FBR, and Super Admin) - each with dedicated dashboards.
  • Built an AI-powered fraud detection engine to automatically identify duplicate ownership claims, fake CNICs, tax evasion, and suspicious transfer patterns.
  • Integrated automated PDF generation for legal documents including Digital Fard, e-Stamp Paper, and Mutation Certificates using PDFKit with QR code verification.
  • Designed and implemented an 11-table relational database with 3 views, 2 stored procedures, and 30+ JWT-secured REST API endpoints.
  • Deployed the full application using Docker and docker-compose with enterprise-grade security (Helmet.js, bcrypt hashing, rate limiting, and SQL injection prevention).
